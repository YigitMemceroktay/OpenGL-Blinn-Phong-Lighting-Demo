# OpenGL-Blinn-Phong-Lighting-Demo

## This application a simple implementation of the 1977 Blinn-Phong lighting model


## Modifiable Values
* Object Color
* Ambient Light Color
* Diffuse Light Color
* Specular Light Color
* Specular Power
* Ambient Power
* Light Rotation
